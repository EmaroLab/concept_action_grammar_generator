This class is used to fully describe the grammar obtained by parsing a source code. Particularly, instances of this class are created during constructors of GrammarCreator.

More in details, this grammar contains the preamble and body ASTs for a given source code, as well as those nodes collected in an java.util.HashMap with node ID has identifier (this is done by using: it.emarolab.cagg.core.language.SyntaxNode#toMap()).

To note that the methods implemented from this class may be inefficient (also because their are designed to be used with the GUI). So, for a real usage of this sistem it is suggest to serialise those results and use them directly without compute them all the times that a result should be evaluated. In order to do so, use: GrammarCreator#serialise(String)

it.emarolab.cagg.core.language.syntax.GrammarPrimitive.GrammarPrimitive ( TextualParser  parser)

This method construct the preamble and body trees and maps from the input parameter. More in details, the preamble and body trees are given by calling: it.emarolab.cagg.core.language.parser.TextualParser#getASPreambleTreeCopy() and it.emarolab.cagg.core.language.parser.TextualParser#getASBodyTreeCopy(). Where a copy is done in order to do not modify all the visualisations of those trees in the GUI during evaluation and manipulation (even if this is not efficient). Finally, this constructors class it.emarolab.cagg.core.language.SyntaxNode#toMap() on those tree in order to fully initialise this class.

Parameters
parserthe source code parser from which this Grammar should be created.

String it.emarolab.cagg.core.language.syntax.GrammarPrimitive.searchMainRuleName ( )

This method (not called on constructor) returns the node data attached to the ASN that defines the main source file and its main rule. To note that it breaks the searching loop when the first occurrence is found. It is important to know that this implementation is not efficient and so it is raccomanded to call it as few times as possible, see also isMainGrammar() and getMainRuleName().

Returns
an empty String if the preamble map does not contains a it.emarolab.cagg.core.language.syntax.abstractTree.syntaxNodeType.StartSyntaxNode node. Otherwise it returns the node data attached to it (e.g. the name of the main rule).
