void it.emarolab.cagg.debugging.baseComponents.customPanel.LogDebuggingGuiPanel.setContents ( )

In this panel the layout of the

contentPanel

is set to

BoxLayout(contentPanel, BoxLayout.PAGE_AXIS)

. Than, it sets the

contentScrollPane

to never scroll horizontally for text line wrap. Finally, adds a javax.swing.JTextArea in which show the logs and set its behavior.
