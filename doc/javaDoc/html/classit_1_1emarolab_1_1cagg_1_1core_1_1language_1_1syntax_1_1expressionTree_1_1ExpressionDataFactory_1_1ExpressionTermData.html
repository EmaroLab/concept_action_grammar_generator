This class implements a node data that contains a String and a Boolean information. Particularly, it extends: it.emarolab.cagg.core.language.BaseDoubleData, with

<String, Boolean>

parameters.

In particular, it is used to describe the information related to a expression term node (EN leaves) it.emarolab.cagg.core.language.syntax.expressionTree.expressionNodeType.TermExpressionNode. In this case, the intance1 (string) contains the name of the term while the instance2 (boolean) describes if it is a term name or a sub rule declaration. More in details, the string literal

name

contains the value of the Term that can be of the type: FROM_IDENTIFIER, FROM_STRING or FROM_DECLARATION. While the boolean flag

isDeclaration

is

true

if the type of this term is

FROM_DECLARATION

;

false

otherwise. See more specifications on the type of term from ExpressionDataFactory.ExpressionTermData#getTokenType(String).
