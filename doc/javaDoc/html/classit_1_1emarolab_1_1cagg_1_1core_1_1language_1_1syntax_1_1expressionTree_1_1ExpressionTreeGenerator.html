This class is used from it.emarolab.cagg.core.language.parser.ANTLRInterface.TreeParserListener in order to create Expression Tree (ET) from a Parsing Tree (PT). Moreover, the created ET is suppose to be the node

data

of a Abstract Syntax Tree (AST).

The purpose of this class is to create an ET during parsing. In particular, it is possible to add operation when they are intercepted by the parser (enter rule) and than update the ET in order to generate it. More in details, by adding operation this class manage a set of

Queues

(implemented in ExpressionQueues). On the other hand, by updating the tree it pull informations for the queues in order to add them in the tree correctly.
