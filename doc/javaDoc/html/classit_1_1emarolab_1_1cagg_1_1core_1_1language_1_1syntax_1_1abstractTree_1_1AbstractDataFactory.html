This class contains useful implementation for the

data

related to each nodes (ASN) of the Abstract Syntax Tree (AST) (it.emarolab.cagg.core.language.syntax.abstractTree.AbstractSyntaxNode). Particularly, this class gives easy cunstructors for objects to be instantiated and returned in it.emarolab.cagg.core.language.syntax.abstractTree.AbstractSyntaxNode#assignData(). Moreover, all the classes that uses this factory are extension of the

AbstractSyntaxNode

and they should be collected in the package: { dotVocal.nuanceGoogleIntegration.semanticParser.parser.abstractSyntaxGenerator.nuanceSyntaxNode.syntaxNodeType}.

More in details, this class implements classes as extensions of it.emarolab.cagg.core.language.syntax.abstractTree.AbstractNodeData and it.emarolab.cagg.core.language.syntax.abstractTree.AbstractNodeDoubleData. Particularly, the node data managed by this factory are: ASNodeEmptyData, ASNodeStrData, ASNodeStrStrData and ASNodeStrTreeData.

The purpose of the classes implemented in this file is to give a flexible interface for AST node modifications and extensions. In particular, each classes that are extending it.emarolab.cagg.core.language.syntax.abstractTree.AbstractSyntaxNode must define the type of data by using those templates or by extending them for extra features. Possible example of this mechanism is shown in class it.emarolab.cagg.core.language.syntax.abstractTree.syntaxNodeType.RuleSyntaxNode; which uses a custom node data class: the

ASNodeStrTreeTagData

.
