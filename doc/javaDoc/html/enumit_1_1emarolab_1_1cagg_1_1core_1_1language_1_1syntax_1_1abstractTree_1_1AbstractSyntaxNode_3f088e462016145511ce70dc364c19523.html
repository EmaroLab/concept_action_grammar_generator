This enumerator defines the possible node types in the Abstract Syntax Tree (AST). Particularly, each nodes that implements it.emarolab.cagg.core.language.syntax.abstractTree.AbstractSyntaxNode must return, as a type (SyntaxNode#assignType()), one different value of this enumeration.
Add value to this enumeration if you want to add a new abstract syntax node type.
