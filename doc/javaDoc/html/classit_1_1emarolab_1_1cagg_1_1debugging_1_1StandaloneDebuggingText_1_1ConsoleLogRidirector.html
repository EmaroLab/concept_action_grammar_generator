This class is used along all the system to have a basing logging object. Particularly, it listen for the standard streams:

System.out

and

System.err

and redirect their messages into the StandaloneDebuggingText.Logger object.
Moreover, the messages can be also printed in the standard java console before to redirect it.
It is important to note that if some exceptions occur in to the system this class may not print the results as long as its buffer is processed. To obviate to this problem and see the longs in the GUI interact in order to process other logs (e.g. open a new grammar dialog and cancel it). While, from API it is possible to call: Logger#flushOut(), Logger#flushErr() or Logger#flushOutErr() to print out all the messages.
