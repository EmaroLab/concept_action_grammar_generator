This enumerator defines the possible node types in the Expression Tree (ET). Particularly, each nodes that implements it.emarolab.cagg.core.language.syntax.expressionTree.ExpressionNodeOld must return, as a type (SyntaxNode#assignType()), one different value of this enumeration.
Add value to this enumeration if you want to add a new Expression Tree node type.
