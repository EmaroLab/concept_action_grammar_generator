This class extends it.emarolab.cagg.debugging.baseComponents.BaseDebugginGuiPanel to show the main complete grammar rule tree. This tree is obtained from all the expression tree nested with respect to the main rule and sub rules. Particularly, this panel shown by tabs a different copied tree for each found solutions.
Moreover, this class change the default title click of its super class by adding a method to easily collapse/expand only the node evaluated as true.
Finally, it also implement the SemanticTreeGuiPanel.MyTreeCellRenderer in order to change the color of the complete tree node with respect to their evaluating values.
