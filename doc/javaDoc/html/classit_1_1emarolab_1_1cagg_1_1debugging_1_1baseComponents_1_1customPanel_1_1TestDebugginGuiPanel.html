This class extends it.emarolab.cagg.debugging.baseComponents.BaseDebugginGuiPanel to allow user to insert input and analyse system results. Particularly, the user input are given in the realted text field as a sentences in order to simulate a speaking situation. While, the generic system results are shown in the text field below.
