This class extends it.emarolab.cagg.debugging.baseComponents.BaseDebugginGuiPanel to show the source text and allows the user to modify it. To node that the parsing procedure used in the GUI always consider the last source text configuration shown in this panel to proceed with the system computations.

contentScrollPane
