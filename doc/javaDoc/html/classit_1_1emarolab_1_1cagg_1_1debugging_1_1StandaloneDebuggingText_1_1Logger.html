This class is used along all the system to have a basing logging object. Particularly, it recieve message from the standard streams:

System.out

and

System.err

basing on the redirector object given on constructor (which is resposible to log those messages also on the Java console or not).
Moreover, it can be used also to directly log data for debugging and showing purposes both in the console or in the GUI.
It is important to note that if some exceptions occur in to the system this class may not print the results as long as its buffer is processed. To obviate to this problem and see the longs in the GUI interact in order to process other logs (e.g. open a new grammar dialog and cancel it). While, from API it is possible to call: Logger#flushOut(), Logger#flushErr() or Logger#flushOutErr()
