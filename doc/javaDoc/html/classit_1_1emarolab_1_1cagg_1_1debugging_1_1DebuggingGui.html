abstract GuiEvaluationInterface it.emarolab.cagg.debugging.DebuggingGui.getEvaluator ( GrammarBase<?>  merger)
abstractprotected

This method is called from DebuggingStaticActioner#testClick(Grammar, TestDebugginGuiPanel, SemanticTreeGuiPanel) and should return a new instance of the evaluator used in the GUI at every call.
By default it returns cagg.core.evaluator.baseInterface.GuiSemanticEvaluator. You can override this method to use a different evaluator on the GUI.

A minimal working implementation is:
protected GUISemanticEvaluator getEvaluator( String inputStr, Grammar merger){
BaseInputFormatter formatter = new InputFormatter( merger, inputStr);
return new GUISemanticEvaluator( formatter);
}

Parameters
formatterthe object formatting the input in tree paths to be evaluated.
mergerthe object contains the parsed grammars
Returns
a new evaluator to be used in the debugging GUI.
