Detailed Description

Version
Concept-Action Grammar Generator 1.1
File: src//cagg.core.sourceInterface.parser.ANTLRInterface.LogicalParserListener.java
Author
Buoncompagni Luca
DIBRIS, emaroLab,
University of Genoa.
15/gen/2016

This class extends the CAGGSyntaxDefinitionBaseListener in order to set the value of flags used later to identify the the meaning of a node in the Parsing Tree (PT) based on its positions with respect to neighbourhood nodes. This object is design to assure flag consistency in

enter

-ing node methods.

Moreover, it is important to know that this implementation is based on the ANTLR grammar definition available in the file:

../files/parserGrammar/NuanceGrammar.g4
