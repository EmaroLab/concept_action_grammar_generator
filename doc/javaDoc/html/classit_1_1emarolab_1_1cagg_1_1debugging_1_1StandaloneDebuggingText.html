This class is used to initialise the Object (it.emarolab.cagg.debugging.StandaloneDebuggingText.ConsoleLogRidirector) that is used to rederect logging messages of all the system to the GUI and Java console.
It is important to be aware that for use the system and its logging features (it.emarolab.cagg.debugging.StandaloneDebuggingText.Logger) thorugh API, the main application function must call:

new DebuggingText();

at the begin of its procedure.
