This class implements a base structure for all the custom panels located in package: it.emarolab.cagg.debugging.baseComponents.customPanel. Thus, it is never intanciated directly.
Those types of panels are aimed to be located into a javax.swing.JSplitPane which is also given as input in the constructor. It is important to remark that this object automatically add it self into its parent.
Moreover, the panels contain a title composed by a button that is used to maximise and minimise the contents in the parent

JSplitPane

.
Also, they contains a javax.swing.JScrollPane focused on a

contentPanel

(javax.swing.JPanel) which is the place where the extending objects should place their contents.

This method does not do any computations and it is called at when the title is clicked by the user. Its purpose is to be overwritten from the extending class in case in which the behavior should be customised. Note that, if parent of this class is a JSplitPnale (thus instantiated by using either: BaseDebugginGuiPanel(JSplitPane, String, Color, Color, Boolean) or BaseDebugginGuiPanel(JSplitPane, String, Color, Color)) this method is called after that the collapsing/expanding behaviour is performed.
